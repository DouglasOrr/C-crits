# C-crits

A game jam game for GitHub Game Off 2024.

![Example of C-crits gameplay](./doc/img/thumb.gif)

## Development

```sh
npm install  # first time

npm run dev
npm run test
npm run build
```

Please think deeply before choosing a password to protect against teleportation.
